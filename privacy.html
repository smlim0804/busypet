Privacy Policy

Last updated: 2026-05-20

TL;DR: We do not collect, transmit, or sell any personal data. No analytics, no trackers, no ads. Everything BusyPet stores stays on your local browser.

What we store locally

BusyPet uses chrome.storage.local (a Chrome-provided per-browser storage) to remember your preferences:

  • Selected pet characters per slot
  • Behavior settings (speed, size, area, mouse mode, effect mode, language)
  • Your custom shortcut bookmarks
  • Saved pet positions per tab
  • Pro license key (if you purchased Pro)
  • Pro trial expiration timestamp (if you started the 5-min trial)

This data never leaves your computer. Uninstalling the extension removes all of it.

What we read

  • CPU and memory usage (chrome.system.cpu, chrome.system.memory): used only to tune pet animation speed. Never transmitted.
  • Active tab URL: used to identify which tab to show the pet on (cross-window follow). Never transmitted or logged.
  • Page favicon: we display the favicon of sites in your shortcut bookmarks via DuckDuckGo's public favicon service (icons.duckduckgo.com/ip3/{domain}.ico) — a single HTTP GET per shortcut, no cookies, no tracking.

Network requests

  • DuckDuckGo favicon service: one image fetch per saved site. No personal data is sent.
  • Pro license validation (only when you click Activate after entering a license key): one HTTPS call to Lemon Squeezy. Only the license key string is sent.

That's the entire list. We do not run analytics, telemetry, error reporting, or A/B testing services.

Permissions explained

  • <all_urls>: required to inject the pet content script on any web page so it can roam across all sites you visit. We do not read page content — we only render an overlay.
  • system.cpu, system.memory: pet animation speed.
  • storage: settings persistence on your device.
  • alarms: schedules the 5-minute Pro trial expiration check.

Children

BusyPet does not knowingly collect personal data from children. It displays purely decorative pixel characters and stores no identity.
