Privacy Policy

Last updated: 2026-05-22 · Extension version 1.0.0

TL;DR: BusyPet does not collect, transmit, or sell any personal data. No analytics, no trackers, no advertising. The extension never reads your page contents — it only renders a pet overlay. All settings stay on your local browser (with optional Chrome account sync for your Pro license key only).

What we store locally

BusyPet uses chrome.storage.local (per-device) to remember:

  • Selected pet characters per slot (1–5)
  • Per-slot behavior settings (motion, mouse mode, speed, size, effect mode, activity area)
  • Custom shortcut bookmarks (the URLs you choose to pin)
  • Saved pet positions per tab
  • UI language preference (ko / en / ja / zh)

What we sync across your Chromes

BusyPet uses chrome.storage.sync only for the Pro license key and 5-minute trial state, so a single purchase activates Pro on every Chrome of your Google account:

  • Pro license key (if you purchased Pro)
  • License activation instance ID
  • Last validation timestamp
  • 5-minute Pro trial expiration timestamp (if started)

This data is synced through your own Google account by Chrome itself. We never see your sync data; only Chrome and the Google account holder can access it.

What we read from your system

  • CPU and memory usage (chrome.system.cpu, chrome.system.memory): used only to tune pet animation speed (the pet runs faster when your system is busy). These values are never transmitted anywhere.
  • Active tab ID (chrome.tabs via the <all_urls> host permission): used solely to identify which tab to show the pet on so the pet follows you across tabs and windows. The tab URL or page content is never transmitted or logged.
  • Page favicon (only for shortcut bookmarks you explicitly add): we display each shortcut's site icon via DuckDuckGo's public favicon service (icons.duckduckgo.com/ip3/{domain}.ico) — a single anonymous HTTP GET per shortcut. No cookies, no tracking, no personal data.

Network requests we make

  • DuckDuckGo favicon service: one image fetch per saved shortcut, only when the shortcut bubble is shown. No personal data is sent — only the public hostname of the bookmark you added.
  • Gumroad license validation (api.gumroad.com): called only when you click the "Activate" button after entering a Pro license key, and once every 7 days for re-validation. We send only the license key string and a generated instance name. We do not send any personal information.

That is the complete list of network requests. We do not run analytics, telemetry, error reporting, crash logging, or A/B testing services.

What we do NOT do

  • We do not read or modify the content of web pages you visit. The content script only injects an overlay element ("the pet") above the page.
  • We do not track which sites you visit, your browsing history, your clicks, your scrolls, or any user activity.
  • We do not use cookies, web storage, or fingerprinting on third-party sites.
  • We do not share, sell, or rent any data to third parties.
  • We do not serve ads.

Permissions explained

  • <all_urls> (host permission): required to inject the pet overlay content script on any web page so the pet can roam across all sites you visit. We use this only to render the overlay — never to read page contents.
  • system.cpu, system.memory: used only to tune pet animation speed locally. Values never leave the device.
  • storage: persists user settings and the Pro license key locally (and the license key syncs through Chrome if Chrome sync is enabled).
  • alarms: schedules the 5-minute Pro trial expiration check and a periodic (every 6 hours) license re-validation ping to Gumroad.

Data retention and deletion

All locally stored data lives in your browser only. Uninstalling the extension removes everything stored by chrome.storage.local. To also remove synced data (license key), use Chrome's sync data dashboard at chrome.google.com/sync or click "Deactivate" inside the extension options before uninstalling.

Children

BusyPet does not knowingly collect personal data from children under 13. The extension displays purely decorative pixel-art characters and stores no identity information.

Third-party services

  • Gumroad (license verification only): we send the license key string to api.gumroad.com when you activate or revalidate. Gumroad's own privacy policy applies to that one HTTPS call: gumroad.com/privacy
  • DuckDuckGo Favicon Service (icons only): public anonymous icon fetches. DuckDuckGo's privacy policy applies: duckduckgo.com/privacy

Changes to this policy

If we ever change what data BusyPet touches, this page will be updated and the "Last updated" date bumped. Material changes will be highlighted in the extension's release notes.

개인정보처리방침 (한국어 요약)

최종 업데이트: 2026-05-22 · 확장 프로그램 v1.0.0

요약: BusyPet은 어떤 개인 데이터도 수집/전송/판매하지 않습니다. 광고/트래커/분석 도구 없음. 페이지 내용을 읽지 않으며 오직 펫 오버레이만 렌더링합니다. 모든 설정은 로컬에 저장되며, Pro 라이선스 키만 Chrome 동기화로 본인 기기 간 공유됩니다.

로컬 저장 데이터

chrome.storage.local에 저장:

  • 슬롯별 선택 캐릭터 (1~5번)
  • 슬롯별 행동 설정 (움직임, 마우스, 속도, 크기, 이펙트, 활동 영역)
  • 사용자가 추가한 바로가기 북마크
  • 탭별 펫 저장 위치
  • UI 언어 (ko/en/ja/zh)

Chrome 계정 동기화 데이터

chrome.storage.sync로 본인 모든 Chrome 간 자동 공유:

  • Pro 라이선스 키 (구매 시)
  • 라이선스 활성화 instance ID
  • 최근 검증 타임스탬프
  • 5분 Pro 체험 만료 시각

이 데이터는 본인의 Google 계정을 통해 Chrome 자체가 동기화합니다. BusyPet은 sync 데이터를 직접 볼 수 없으며, 본인과 Chrome만 접근 가능합니다.

시스템에서 읽는 정보

  • CPU/메모리 사용량: 펫 애니메이션 속도 조절에만 사용. 외부로 전송되지 않음.
  • 활성 탭 ID (chrome.tabs): 펫이 어느 탭에 표시될지 식별하기 위해 사용. 탭의 URL이나 페이지 내용은 전송/기록되지 않음.
  • 사이트 파비콘: 본인이 추가한 바로가기 북마크에 대해서만 DuckDuckGo 공개 파비콘 서비스에서 아이콘을 가져옴. 익명 HTTP GET 1회.

외부 네트워크 호출

  • DuckDuckGo 파비콘: 바로가기 말풍선 표시 시 1회씩.
  • Gumroad 라이선스 검증(api.gumroad.com): Pro 키 활성화 시 1회, 그 후 7일마다 재검증 1회. 라이선스 키 문자열만 전송, 개인정보는 전송되지 않음.

안 하는 것

  • 페이지 내용을 읽지 않음 (오버레이만 렌더링)
  • 방문 사이트, 검색 기록, 클릭, 스크롤, 사용자 활동 추적 안 함
  • 제3자 사이트에서 쿠키/저장소/지문 사용 안 함
  • 데이터 판매/공유/제3자 전송 안 함
  • 광고 노출 안 함

권한 설명

  • <all_urls>: 펫 오버레이를 모든 웹페이지에 표시하기 위해 필요. 페이지 내용은 읽지 않음.
  • system.cpu, system.memory: 펫 속도 조절용. 외부 전송 없음.
  • storage: 설정/라이선스 저장 (sync 시 Chrome으로만 동기화).
  • alarms: 5분 체험 만료 + 6시간마다 라이선스 재검증 스케줄링.

데이터 삭제

확장 프로그램 제거 시 chrome.storage.local 데이터는 자동 삭제됩니다. 동기화된 라이선스 정보까지 지우려면 옵션 페이지에서 "비활성화"를 먼저 누르거나 chrome.google.com/sync에서 동기화 데이터를 관리하세요.
